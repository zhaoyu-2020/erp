订单的状态，新订单--收定金--已采购--待发运--已报关--已发运--已收尾款--已结算

销售订单的状态分为，新建/已收定金/已采购/待发运/已发运/已收款/已完成。

销售订单，状态划分
1.创建一个订单后，状态为新建
2.在财务系统里收款之后，销售订单状态可以改为已收定金
3.当有至少一条采购合同的时候，可以改为已采购，已采购时校验销售合同和采购合同的总数量，差太多会弹提示
4.当所有采购合同的状态为待发货时，状态改为待发运
5.当完成报关，货物发走时，状态改为已发运
6.收款完成时，状态改为已收款，已收款时会校验实际金额，和总收款金额，做对比，然后有个提示
7.管理员会点击已结算，则归档

采购订单的状态分为 新建/生产中/待发货/已完成。
1创建一个订单后，状态为新建。
2创建完订单明细之后，可以改为生产中。
3可以提货了，改为待发货。
4发完货可以已完成，完成时，要检查材质单，发票等。
