首先创建外贸订单
根据订单，创建订单明细
有订单之后，创建收款（客户收款，美金）
采购根据订单明细执行采购，这里要有一个状态提醒采购新的有采购需求
真实采购完成后，有了真实的数据，完善订单明细。

货物的定金，尾款，各项目的结算重量和金额，运费，加工费

确定汇率
登记收尾款。
登记海运费，登记地面费用。（CIF涉及）

之后就有这个订单的结算报表了。

订单的状态，新订单--收定金--已采购--待发运--已报关--已发运--已收尾款--已结算
